What We Made

As part of our internet of Things class we were tasked to create an Alexa app. After touring JUMP Tyler suggested that we give Mango, a flamingo sculpture, a voice. We have created an app that allows users to ask Mango questions about flamingos, although we would like to one day expand the conversations that mango can have.
